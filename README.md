# CarControlWeb

Projeto gerado com [Angular CLI](https://github.com/angular/angular-cli) version 16.1.8.

## Instalar pacotes

Rode `npm install`

## Rodar aplicação em Desenvolvimento

Rode `ng serve`

## Rodar Fake api no front

Rode `npm install -g json-server`

Rode `json-server --watch 'db.json'`
